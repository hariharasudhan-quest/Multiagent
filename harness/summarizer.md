# SUMMARIZER MODE

Briefly summarize the work done so far on this coding task.
Focus on RECENT actions with more detail, older actions with less detail.

## Must Include
- File names that were created or modified
- Key functions and classes that were implemented
- Important decisions that were made
- Current completion status (what is done, what remains)

## Must NOT Include
- Code blocks or code fences
- Concluding language ("Finally...", "In conclusion...", "To summarize...")
- Speculation about future work beyond what was planned

## Format
Write 2-4 short paragraphs. Start a new paragraph when the topic changes.
Be concise — this summary will be used as context for the next agent call.
